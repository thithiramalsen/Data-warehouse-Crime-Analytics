# Power BI Dashboards

This folder is intended to contain Power BI Desktop files (`.pbix`) and exported report artifacts.

Guidelines:

- Connect reports to the local CSV at `Data Warehouse/Datasets/Crime_Data_from_2020_to_2024_20260326.csv` or to the warehouse database rather than embedding raw data into the report file when practical.
- When sharing `.pbix` files, include `DATASET_SOURCE.txt` and any connection instructions to help others reproduce the data refresh.

If you want, I can add a template `.pbix` with sample visuals and a readme showing how to set the data source to the local CSV.
