# Excel Analysis Workbooks

This folder is intended to contain Excel workbooks, pivot tables, and lightweight analysis artifacts created during exploration and reporting.

Guidelines:

- Do not embed the full raw CSV inside workbooks. Instead, connect to the dataset at `Data Warehouse/Datasets/Crime_Data_from_2020_to_2024_20260326.csv` or to a local database built from the data warehouse.
- Keep derived tables and outputs small and add them to `.gitignore` if large.

If you'd like, I can add a sample pivot workbook template that reads from the CSV.
